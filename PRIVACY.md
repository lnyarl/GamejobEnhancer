# 개인정보 처리방침

**Gamejob Enhancer**는 사용자 개인정보를 수집, 저장, 전송하지 않습니다.

## 수집하지 않는 정보

- 외부 서버로 어떠한 데이터도 전송하지 않습니다.
- 사용자 활동, 입력 내용, 클릭/스크롤 이력, 페이지 콘텐츠를 추적하지 않습니다.
- 광고 식별자, 분석 도구(Google Analytics, Mixpanel 등)를 사용하지 않습니다.
- 개인 식별 정보(이름, 이메일, 연락처 등)에 접근하지 않습니다.

## 로컬에 저장하는 정보

`chrome.storage.sync` API를 사용해 다음 두 가지를 저장합니다:

1. **사용자 설정값** — 옵션 페이지에서 선택한 활성화 여부, 폰트 패밀리
2. **채용관 섹션의 접힘/펼침 상태** — 페이지 새로고침 후에도 유지하기 위함

이 정보는 Chrome 브라우저가 사용자 본인의 Google 계정에 저장하며, 같은 계정으로
로그인된 본인의 다른 기기에서만 동기화됩니다. 확장 프로그램 개발자 또는
제3자는 이 정보에 접근할 수 없습니다.

## 권한 사용

- `storage` — 위에 명시한 사용자 설정값 저장 용도로만 사용합니다.
- `host_permissions: https://www.gamejob.co.kr/*` — gamejob.co.kr 도메인의
  페이지에 UI 개선 콘텐츠 스크립트를 주입하기 위함입니다. 이 도메인 외에는
  접근하지 않으며, 사이트의 콘텐츠를 외부로 전송하지 않습니다.
